# 英语学习资源汇总

## 核心理念：可理解输入 + 沉浸式学习

**i+1 理论**（Stephen Krashen）：学习者在语境中接触"略高于当前水平"的语言输入时，语言习得效率最高。视频 + 母语解释正是这个理论的最佳实践——用中文辅助理解英文，让大脑在真实语境中内化，而非机械记忆。

**核心方法：影子跟读（Shadowing）**
听到一句英文 → 暂停 → 立刻复述 → 重复直到接近原声。训练口语反应速度和语感。

## 四、学习方法优先级

### 快速见效路径（适合时间有限的学习者）

1. **每天 20 分钟 VoiceTube** — 选一个感兴趣的主题（美食/科技/旅行），开双语字幕，先看中文理解，再看英文
2. **影子跟读** — 每条视频选 3 句，暂停→复述→重复，直到能跟上语速
3. **每周 1 个生活场景**（如点餐、机场、购物），集中积累该场景词汇

### 长期积累路径

1. **BBC Learning English 合集** — 系统学习发音、语法、词汇
2. **Shadowing + 录音对比** — 用手机录自己的发音，和原声对比
3. **写英文日记** — 用 DeepL 批改，逐步减少对中文的依赖

## 五、按水平推荐

| 水平 | 推荐资源 | 每天建议 |
|---|---|---|
| **入门（A1~A2）** | BBC Learning English / Shane 精听 / 瑞秋英语发音课 | 15~30 分钟 |
| **中级（B1~B2）** | VoiceTube / English with Lucy / VibeEnglish | 30~45 分钟 |
| **高级（B2~C1）** | YouTube 全英内容（不加字幕）/ 英美剧生肉 | 30~60 分钟 |

## 六、关键原则

> **不要追求"看懂每一个词"** — 专注理解大意，在语境中猜测词义，这是母语习得的自然方式。
>
> **高频重复 > 长时间低效** — 每天 20 分钟专注练习，远好过周末 3 小时集中学习。
>
> **输出倒逼输入** — 找机会说英语（哪怕自言自语），比只听不说进步快得多。
