## User Question: Open Apps by Name?
`termux-open` cannot directly open apps by name. It accepts:
- A file path → opens with system handler
- A URL (https://, tel:, mailto:, etc.) → opens in appropriate app
- No generic "app name" resolution

Workaround: need to know the app's URL scheme (e.g. `weixin://` for WeChat) or use `termux-open-url` with an intent URL.
